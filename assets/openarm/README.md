# OpenArm robot assets

The meshes under `openarm_description/` originate from
[`enactic/openarm_description`](https://github.com/enactic/openarm_description)
and were verified byte-for-byte against commit
`6c7b720f1ba48e8bafa3a3dc752c45f397b42221`.

`urdf/openarm_v1.urdf` is a generated OpenArm v1 description retained for
HandUMI's kinematic simulator. It has local gripper-limit corrections, so it
is not claimed to be an unmodified upstream artifact.

The upstream description package is Apache-2.0, Copyright 2025 Enactic, Inc.
Its license is bundled in `LICENSE.openarm`.
